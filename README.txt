== Description ==

#### Raileo Monitoring for Wordpress helps you to check your website downtime, ssl validity, pagespeed monitoring and other insights.

Raileo monitoring for wordpress helps you to read your monitoring status from Raileo Dashboard.

This plugin helps you to read and visualize the monitoring data from Raileo about your URLs. Uptime data, SSL expiry and pagespeed can be
visualuzed using this plugin.

#### Features

- Connect with your Raileo account
- Show all the URLs in your Raileo Dashboard
- Following data can be seen for each URL
    - Uptime data
    - SSL Expiry days and date
    - Pagespeed scan data (Website performance score, seo score, accessibility score, best practices and PWA score)
- Developer friendly API is provided. Documentation on the API can be found [here](https://docs.raileo.com/api)


#### What is Raileo?

Raileo is a Website monitoring service. Raileo helps you to monitor downtime for your service, monitor when your SSL expires and using Google 
lighthouse measure your website's performance and other metrics.
Raileo comes with a free plan to help you get started.
If you are not using Raileo yet, [create an account under 30 seconds](https://raileo.com/register)

== Installation ==

#### Installing the plugin
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Raileo** and click "*Install now*"
2. Alternatively, download the plugin and upload the contents of `raileo.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
3. Activate the plugin
4. Set [your API key](https://raileo.com/admin/api_keys) in the plugin settings.

#### Configuring Raileo monitoring
1. Go to [Raileo Website](https://raileo.com)
2. Add your website URLs in the [dashboard](https://raileo.com/admin/urls)
3. Come back to your wordpress plugin and refresh, you can see the URL in the plugin
4. Click on the URL to see more information on your URL such as monitoring data.
